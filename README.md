# facelessfreedom

Static recreation of a Swipio-style landing page using plain HTML, CSS, and JavaScript.

## Run locally

Use any static server from this folder, for example:

```bash
python3 -m http.server 5173
```

Then open <http://localhost:5173>.

## Included

- Sticky navigation + anchor links
- Hero section with interactive swipe-card phone mock
- Feature grid
- How-it-works steps
- Auto-rotating testimonial slider
- FAQ accordion
- Download CTA section and footer
- Responsive styles for desktop/mobile
